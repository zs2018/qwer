HTML5 基础学习

前言

  1. 认识HTML5
    1.什么是HTML5: HTML5是新一代开发web富客户端的整体解决方案，他是包含HTML+CSS+Javascript整体提升的心一代web应用开发平台
    1. 为什么要学习HTML5:在HTML5时代能够开发很多富客户端应用，能够开发更多极具表现力的网页，能够做出PC端，APP端，微信端的网页应用程序，能够开发混合本地应用和游戏等等，应用非常广泛（能赚钱拿高薪）
    2. 如何学习HTML5: 先从基本的HTML新增的各种标签开始学习，然后学习javascriptAPI,在学习新增的CSS3各种属性
    3. 总结HTML5时代的我们需要学习很多新的的东西，能够开发各种现代的web应用程序，能拿高薪
  2. HTML5的语法规范
    1. HTML5语法规范：emmet快捷键html:5 相对之前的html而言语法规范更加简洁 更宽松（但实际开发中要是要严格要求自己）

语义化

  1. 什么是语义化： 就是要使HTML标签具有更好的可读性，可以清晰的传达每个标签要表达的意思，以便于被更好的处理和解析
  2. 为什么要学习语义化：为了能够更好的表达网页传达的信息，更好的被浏览器处理和解析，为了搜索引擎优化等等
  3. 如何学习语义化：
    1. 了解过去HTML4对语义化的处理
    2. 学习HTML5新增的语义化标签
      1. 新增的语义化标签有哪些： nav header footer section aside article process (其中nav header footer section 重点掌握)
    3. 处理低版本IE浏览器的兼容性问题
      1. 通过document.createElement() 的方式其创建新标签能够被低版本的IE浏览器支持
      2. 可以通过引入第三方的包html5shiv实现兼容
  4. 总结：HTML5新增了很多语义化标签解决了传统需要用类名来实现语义化的问题，同时规范了语义化标签的具体写法，让语义化更加简单

智能表单

  1. 什么是智能表单:比传统的表单更加只能，功能更加强大的表单
  2. 为什么要学习智能表单:因为传统的表单已经满足不了我们现代web开发特别是移动web开发的需求，页面上需要更多的新表单来实现更好的用户体验
  3. 如何学习智能表单：
    1. 输入类型表单：email tel number search url range color time date datetime month week(重点掌握email tel number search url)
    2. 表单元素： datalist keygen output(了解即可开发中用得都比较少)
    3. 表单属性： placeholder autofoucus mutiple autocomplete form novalidate requied pattern(都有可能用到 但是频率不是很高)
  4. 总结： HTML5的在表单这一方面新增了很多表单类型，表单元素，表单属性，让表单处理更加高效，特别是移动端用户体验和展现效果很好，移动端的使用要优于PC端

HTML5多媒体

  1. 什么是HTML5多媒体： 就是HTML5的音频播放，和HTML5视频播放
  2. 为什么要学习HTML5多媒体：因为传统要在网页上实现音频和视频播放需要依赖第三方组件flash,flash需要用户安装一个插件，使用起来相对比较麻烦，而且苹果设备不支持flash导致丢失用户群体，而HTML5的视频和音频不想要第三方插件而且兼容性跨平台性好，所以使用HTML5的多媒体
  3. 如何学习HTML5的多媒体：
    1. 学习HTML5音频播放： audio标签 里面有一些基本属性
      1. controls： 控制器 控制音频的开始和暂停的控制器
      2. loop: 循环 控制音频的循环播放次数
      3. autoplay: 自动播放 控制音频是否自动播放
    2. 学习HTML5视频播放： video标签
      1. controls： 控制器 控制视频的开始和暂停的控制器
      2. loop: 循环 控制视频的循环播放次数
      3. autoplay: 自动播放 控制视频是否自动播放
      4. width: 宽度 控制视频区域的宽度
      5. height: 高度 控制视频区域的高度 但是宽度高度不会拉伸视频区域而会取最小值
    3. 处理音频和视频的兼容性问题：
      1. 在视频或者音频标签里面添加多个source标签可以放置多个视频和音频文件，达到各种浏览器的支持
  4. 总结：HTML5多媒体解决了传统网页多媒体依赖第三方组件flash的问题，同时使用起来更加方便，应用比较广泛，例如网页音频视频播放器，网站的一些小广告，背景音乐等等

DOM扩展

  1. 什么是DOM扩展：就是在原有的DOM操作属性方法上，新增了一些DOM操作的属性方法例如DOM的增删该查之类等等操作
  2. 为什么要学习DOM扩展：因为传统的JS在DOM操作上比较繁琐，比较麻烦，所以后来出了jQuery解决了这个问题，但是现在的HTML5又把DOM操作变得简单而且接近jquery所以也用得比较多
  3. 如何学习DOM扩展：
    1. 获取元素：
      1. 根据类名获取所有元素：document.getElementByClassName()
        返回伪数组
      2. 根据CSS选择器获取第一个元素：document.querySelector()
        返回第一个元素
      3. 根据CSS选择器获取所有元素：doucment.querySelectorAll()
        返回伪数组
    2. 类名操作：
      1. Node.classList.add() 给当前节点添加一个类名
      2. Node.classList.remove() 给当前节点删除一个类名
      3. Node.classList.toggle()
        给当前节点替换一个类名 有则删除，无责添加
      4. Node.classList.contains()
        查询这个类名是否在当前节点存在 存在返回true 不存在返回false
    3. 事件监听：
      1. Node.addEventListener(event,function,useCapture)
        添加事件监听，给当前节点添加一个事件监听，
        和传统的绑定事件的方式的区别就是 这种方式可以添加多个事件
        可以设置事件的捕获和冒泡 useCapture默认为false表示取消事件捕获（则为事件冒泡的过程调用回调函数）
        useCaptrue设置为treu表示开启事件捕获（则为事件捕获的过程中调用回调函数）
      2. Node.removeEventListener(event,function,useCapture)
        删除事件监听
    4. 自定义属性：
      1. 设置自定义属性：
        在标签里面加上 data-*=”属性值” 的方式添加自定义属性
      2. 获取自定义属性的值：
        1. 使用getAttribute(“data-属性名”) 的方式可以获取自定义属性的值 需要带data-
        2. 使用dataset[‘属性名’]的方式获取自定义属性的值 不需要带data-
      3. 设置自定义属性的值：
        1. 使用setAttribute(‘data-属性名’,”值”); 需要带data-
        2. 使用dataset[‘属性名’] = “值”; 不需要带data-
  4. 总结：HTML5在DOM操作上扩展了很多东西，简化了DOM的操作应用很广泛特别是获取元素，添加事件监听以后经常会用到，要熟练使用
